Lost in Reminiscene - Roblox Studio Game Project

∣ Description ประวัติความเป็นมา

Project เกมที่ผมทำขึ้น ในเวลาว่าง

ผมเริ่มสร้างเกมนี้จากความสนใจในด้านการพัฒนาเกมและต้องการท้าทายความสามารถของตัวเองในการสร้างสรรค์สิ่งใหม่ๆ ด้วย Roblox Studio ซึ่งต้องใช้ทักษะในการเขียนโค้ดภาษา LuaU เพื่อสร้างฟีเจอร์ที่น่าสนใจและให้ผู้เล่นได้สัมผัสประสบการณ์ที่ตื่นเต้น ผมได้เรียนรู้วิธีการออกแบบเกม การแก้ไขปัญหา และการปรับแต่งเกมให้เข้ากับความต้องการของผู้เล่น เพื่อให้เกมนี้สามารถพัฒนาต่อไปในอนาคตได้

ตัวอย่าง Code ที่เขียน : LuaU

โดยตัวเกมจะยังอยู่ในเวอร์ชั่นระหว่างการทดลองให้เล่นได้ในช่วงเดือนตุลาคม - ธันวาคม
